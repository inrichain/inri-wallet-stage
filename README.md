# INRI Wallet Stage

Stable wallet build focused on the parts that can work end-to-end right now.

## Working now
- create wallet
- import wallet from seed phrase
- unlock existing wallet
- receive screen with QR code
- send native INRI
- send ERC-20 tokens on INRI
- token list with custom token support
- activity history saved locally
- theme, language, avatar
- PWA manifest/assets

## Still placeholder until real contract integration is added
- Swap screen
- Bridge screen

Those two screens are UI previews only. To make them real, the app still needs the exact router / bridge contract flow, ABI, method names, slippage rules, destination chain handling, and confirmation flow.
